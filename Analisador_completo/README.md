# Analisador Completo

Programa desenvolvido em Python que coleta informações de quatro pessoas e realiza uma análise estatística simples do grupo.

## 📋 Descrição

O programa solicita o nome, a idade e o sexo de quatro pessoas. Ao final, exibe informações calculadas a partir dos dados informados.

As análises realizadas incluem:

* Média de idade do grupo;
* Nome e idade do homem mais velho;
* Quantidade de mulheres com menos de 20 anos.

## 🚀 Funcionalidades

* Cadastro de quatro pessoas.
* Cálculo da média de idade.
* Identificação do homem mais velho.
* Contagem de mulheres com menos de 20 anos.
* Exibição organizada dos resultados.

## 🛠️ Tecnologias utilizadas

* Python 3

## ▶️ Como executar

1. Clone este repositório.
2. Execute o arquivo principal:

```bash
python Analisador_completo.py
```

3. Informe os dados solicitados para cada pessoa.
