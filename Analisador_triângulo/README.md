# Classificador de Triângulos

Programa desenvolvido em Python que verifica se três segmentos de reta podem formar um triângulo e, em caso positivo, identifica sua classificação.

## 📋 Descrição

O programa recebe os comprimentos de três lados informados pelo usuário e verifica se eles satisfazem a condição de existência de um triângulo.

Caso seja possível formar um triângulo, o programa informa se ele é:

* Equilátero
* Isósceles
* Escaleno

Também realiza uma validação para impedir que valores iguais ou menores que zero sejam utilizados.

## 🚀 Funcionalidades

* Leitura de três segmentos de reta.
* Aceita números com vírgula ou ponto como separador decimal.
* Validação de valores inválidos.
* Verificação da condição de existência de um triângulo.
* Classificação do triângulo.

## 🛠️ Tecnologias utilizadas

* Python 3

## ▶️ Como executar

1. Clone este repositório.
2. Execute o arquivo principal:

```bash
python Classificador_Triangulos.py
```

3. Informe os três lados do triângulo quando solicitado.
