 Verificador de Alistamento Militar

Programa desenvolvido em Python que calcula a idade do usuário com base no ano de nascimento e informa sua situação em relação ao alistamento militar obrigatório.

 📋 Descrição

O programa utiliza o ano atual do sistema para calcular a idade do usuário e informa uma das seguintes situações:

- Ainda não chegou a idade de alistamento;
- Está na idade de se alistar;
- Já passou da idade de alistamento.

Além disso, o programa informa:

- Quantos anos faltam para o alistamento;
- Em que ano o alistamento deverá ocorrer;
- Ou há quantos anos o prazo foi ultrapassado.

 🚀 Funcionalidades

- Cálculo automático da idade.
- Identificação da situação do alistamento.
- Exibição do ano atual.
- Cálculo do tempo restante para o alistamento.
- Cálculo do atraso no alistamento.
- Indicação do ano em que o alistamento deveria ter sido realizado.

 🛠️ Tecnologias utilizadas

- Python 3
- Módulo `datetime`

 ▶️ Como executar

1. Clone este repositório.

```bash
git clone https://github.com/seu-usuario/verificador-alistamento-militar.git
```

2. Execute o programa.

```bash
python main.py
```

3. Informe o ano de nascimento quando solicitado.
