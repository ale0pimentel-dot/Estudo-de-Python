Classificador de Categorias de Atletas

Programa desenvolvido em Python que classifica um atleta em uma categoria esportiva com base na sua idade.

📋 Descrição

O programa solicita o ano de nascimento do atleta, calcula sua idade utilizando o ano atual do sistema e informa a categoria correspondente.

As categorias são:

Mirim: até 9 anos.
Infantil: de 10 a 14 anos.
Júnior: de 15 a 19 anos.
Sênior: de 20 a 25 anos.
Master: acima de 25 anos.
🚀 Funcionalidades
Cálculo automático da idade.
Utilização do ano atual do computador.
Classificação automática da categoria do atleta.
🛠️ Tecnologias utilizadas
Python 3
Módulo datetime
▶️ Como executar
Clone este repositório.
Execute o programa:
python Classificador_Atletas.py
Informe o ano de nascimento do atleta.
