Calculadora de Média Escolar

Programa desenvolvido em Python que calcula a média de um aluno com base em quatro notas e informa sua situação acadêmica.

📋 Descrição

O programa solicita quatro notas, calcula a média aritmética simples e informa a situação do aluno conforme os seguintes critérios:

Reprovado: média inferior a 5,0.
Recuperação: média entre 5,0 e 6,9.
Aprovado: média igual ou superior a 7,0.

O programa também aceita notas informadas com vírgula como separador decimal.

🚀 Funcionalidades
Leitura de quatro notas.
Aceita números com vírgula ou ponto.
Cálculo da média aritmética simples.
Exibição da situação do aluno.
🛠️ Tecnologias utilizadas
Python 3
▶️ Como executar
Clone este repositório.
Execute o programa:
python Media_Escolar.py
Informe as quatro notas do aluno.
