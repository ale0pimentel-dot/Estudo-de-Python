# Simulador de Aprovação de Empréstimo

Programa desenvolvido em Python que simula a análise de um financiamento imobiliário com base no valor do imóvel, salário do comprador e prazo de pagamento.

## 📋 Descrição

O programa calcula o valor da prestação mensal de um financiamento e verifica se ela ultrapassa 30% do salário informado.

Caso a prestação seja superior a esse limite, o financiamento é recusado. Caso contrário, o empréstimo é aprovado.

## 🚀 Funcionalidades

* Leitura do valor do imóvel.
* Leitura do salário do comprador.
* Leitura do prazo de pagamento em anos.
* Conversão automática de valores monetários digitados com vírgula.
* Cálculo da prestação mensal.
* Aprovação ou reprovação do financiamento.

## 🛠️ Tecnologias utilizadas

* Python 3

## ▶️ Como executar

1. Clone este repositório.
2. Execute o programa:

```bash
python Aprovando_Emprestimo.py
```

3. Informe:

   * Valor do imóvel;
   * Salário;
   * Quantidade de anos para pagamento.
