# Verificador de Ano Bissexto

Programa desenvolvido em Python que verifica se um determinado ano é bissexto utilizando as regras do calendário gregoriano.

## 📋 Descrição

O programa solicita ao usuário um ano para análise. Caso seja informado o valor **0**, o sistema utiliza automaticamente o ano atual do computador.

Após a leitura do ano, o programa verifica se ele é bissexto e informa o resultado ao usuário.

## 🚀 Funcionalidades

* Verificação de anos bissextos.
* Utilização automática do ano atual quando o usuário informa `0`.
* Aplicação das regras oficiais do calendário gregoriano.

## 🛠️ Tecnologias utilizadas

* Python 3
* Módulo `datetime`

## ▶️ Como executar

1. Clone este repositório.
2. Execute o programa:

```bash
python Ano_bissexto.py
```

3. Informe o ano desejado ou digite **0** para utilizar o ano atual.
