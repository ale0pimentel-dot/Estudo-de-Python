
Aumento Salarial

Programa desenvolvido em Python que calcula o novo salário de um funcionário com base em uma política de reajuste salarial.

📋 Descrição

O programa solicita o salário atual do funcionário e aplica um reajuste conforme a seguinte regra:

Salários de até R$ 1.250,00 recebem aumento de 15%.
Salários acima de R$ 1.250,00 recebem aumento de 10%.

Ao final, o programa informa o salário anterior e o novo salário após o reajuste.

🚀 Funcionalidades
Leitura do salário atual.
Cálculo automático do percentual de reajuste.
Exibição do salário antigo e do novo salário.
🛠️ Tecnologias utilizadas
Python 3
▶️ Como executar
Clone este repositório.
Execute o programa:
python Aumento_Salarial.py
Informe o salário do funcionário.
