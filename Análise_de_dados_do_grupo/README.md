# Análise de Dados do Grupo

Programa desenvolvido em Python que realiza o cadastro de pessoas e apresenta estatísticas com base nas informações informadas.

## 📋 Descrição

O programa permite cadastrar várias pessoas, registrando idade e sexo. Ao final, apresenta um resumo com informações estatísticas sobre os dados cadastrados.

As estatísticas incluem:

* Quantidade de pessoas com mais de 18 anos.
* Quantidade de homens cadastrados.
* Quantidade de mulheres com menos de 20 anos.

O usuário pode escolher continuar cadastrando pessoas ou encerrar o programa a qualquer momento.

## 🚀 Funcionalidades

* Cadastro contínuo de pessoas.
* Validação da entrada do sexo (`M` ou `F`).
* Validação da opção de continuar (`S` ou `N`).
* Contagem de pessoas maiores de 18 anos.
* Contagem de homens cadastrados.
* Contagem de mulheres com menos de 20 anos.
* Exibição de um relatório final.

## 🛠️ Tecnologias utilizadas

* Python 3

## ▶️ Como executar

1. Clone este repositório.
2. Execute o programa:

```bash
python Analise_Dados_Grupo.py
```

3. Informe a idade, o sexo e escolha se deseja continuar cadastrando pessoas.
